Welcome to corredor's documentation!
====================================

Corredor is a framework for easily building distributed test harnesses. There are generally two
reasons you might want to consider using it:

A) To create bindings for your runner in different languages.
B) To inter-changeably distribute tests across threads, processes or machines.
